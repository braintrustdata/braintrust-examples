# Running

## Pre-requisites

- Node.js >= 18.6.0
- OpenAI API key
- BrainTrust api key

## Install dependencies

```bash
npm install
```

## Build

```bash
npm run build
```

##
Add your OpenAI and BrainTrust API keys to the .env file. Then, load the env file.

```
source .env
```

## Run

```bash
npm run script
```
