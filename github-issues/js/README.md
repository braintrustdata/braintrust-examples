# Running

## Pre-requisites

- Node.js >= 16.6.0
- OpenAI API key
- Braintrust api key

## Install dependencies

```bash
npm install
```

## Build

```bash
npm run build
```

## Run

```bash
OPENAI_API_KEY=<your key> BRAINTRUST_API_KEY=<braintrust key> BRAINTRUST_API_URL=<your braintrust instanse url> node dist/index.js
```
